## available information
### channels
- display name (de/fr/it)
- channel icon (from TV7 playlist)

### programms
- title
- start & end time
- duration
- description
- icon (tele.ch only)
- subtitle
- production country
- credits (tele.ch only)
- episode (season and episode number)
- first release / production year
